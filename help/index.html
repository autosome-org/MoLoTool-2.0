Select Motifs

Transcription Factor Binding Sites (TFBS) motifs are taken from HOCOMOCO. TFBS motifs can be added to the list of selected models using search . The placeholder displays the available input example.

  1. Select collection: the motif collection can be selected using Collection Select Menu near the search window.
  2. Select motif: TF binding motifs can be searched by motif ID, family, subfamily and gene symbol.

The sequences can be entered both i plain text and in FASTA format. Multiple sequences can only be entered in FASTA format.

Adjust P-Value

The desired P-value threshold is shown in both linear and logarithmic scale and restricted to the [ 1.00e-6, 5.00e-2 ] interval. There are three ways to set the desired P-value threshold.

  • Enter precise value: you can enter the desired P-value directly into the textarea where it is shown.
    • Enter logarithmic P-value: the value entered on the right will be -log10(P-value).
    • Enter linear P-value: the value entered on the left will be the exact P-value.
  • Drag P-slider: the scale of the P-Slider is logarithmic, -log10(P-value).

Note, that at lower thresholds you may experience a minor slowdown when visualizing multiple TFBS in long sequences (100bp+).

Sequence Tabs & Motif Controls

If there are no parsing errors, one Sequence Tab is created for each sequence. Sequences with markup and highlighting of the motif hits appear in the Output Window, as in the Demo example above.

You can always change the selected sequence color with the Color Picker. Click on the box with the selected color in the upper left corner of Motif Control to open Color Picker. Newly added motifs use the default color palette.

  • Close Sequence Tab: clickcloseto hide the sequence data from the Output Window and the Motif Hits Table.
  • Copy sequence: click content_copy to copy the sequence with markup and highlighting of motif hits. It remains colored when pasted into popular text editors such as Pages, MS Word, and LibreOffice.
  • Lock scroll: click lock_open to lock the horizontal scroll at the current position. Click it again to unlock.

Horizontal scroll is only available in the Multiple Mode. Therefore, there is no lock_open button in the Single Mode. Read more about the View Modes below.

Motif Hits Highlighting

Motif hit highlighting is a special feature that allows you to better navigate the output and improves the visual experience. It highlights the relationship between the data from Motif Controls, the Output Window, and the Motif Hits Table. Hover over the following elements to trigger the Motif Hits Highlighting.

  • Motif Control: highlight entries of the motif both in the Output Window and the Motif Hits Table.
  • Output Window: hover over a colored hits in the sequence to highlight the corresponding rows in the Motif Hits Table and the respective motif in the Motif Control.
  • Motif HIts Table: hover over a row to highlight the respective motif hit in the Output Window and corresponding motif in the Motif Control.

View Modes

There are two View Modes, Multiple Mode and Single Mode. Multiple Mode is activated automatically after the first data submission. Further submissions activate the last used View Mode.

  • Multiple Mode: shows all the entered sequences without line wrapping. Horizontal scrolling is available when there is not enough space.
  • Single Mode: shows only one selected sequence in the entire Output Window with automatic line breaks.

You can switch between these modes using the Mode Switch located above the right corner of the text input field or Output Window. You can also click on Mode Button / in Sequence Tabs.


If the Single Mode is activated by the Mode Switch, the first sequence is automatically selected to be displayed in Single Mode. If the Single Mode is activated by the Mode Button in the Sequence Tab, that sequence is auto-selected. You can change your selection by clicking another Sequence Tabs.

Motif Hits Table

Detailed data on detected binding sites are provided in the Motif Hits Table at the bottom of the page. Generating the population of this table is the most time-consuming operation.

Clicksyncto invert logo.

The Motif Hits Table has additional controls. Click to select information to be displayed on the table. Selected columns are highlighted in white.

The Motif Hits Table contents can be exported using Export Buttons. Available formats: plain text (copy to clipboard), xlsx , PDF , and TSV .

The Motif Hits Table contains only sites from displayed sequences. According to the chosen View Mode, the table will contain different sites.
