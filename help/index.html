Select models

Transcription Factor Binding Sites (TFBS) models are taken from Human Comprehensive Model Collection (HOCOMOCO). TFBS models can be added to the Model List using search . The placeholder displays the available input example.

  1. Select collection: the required collection of TFBS models can be selected using Collection Select Menu near the search window.
  2. Select model: TFBS models can be found by motif ID, family, subfamily and gene.

Enter sequences

Sequence can be entered both directly and in FASTA format. Multiple sequences can only be entered as FASTA.

  • Enter sequences manually: input sequence or FASTA in text field in the centre then click .
  • Upload a file: click Chose a file in Upload Container (shown bellow) or drag a file directly into it, then the mode will be automatically changed to the View Mode.

Adjust P-Value

P-value is shown in both linear and logarithmic scale and restricted to the [ 1.00e-6, 5.00e-2 ] interval. There are three ways to set the desired P-value.

  • Enter precise value: you can enter desired P-value directly into the area where it is shown.
    • Enter logarithmic P-value: value entered on the right will be -log10(P-value).
    • Enter linear P-value: value entered on the left will be exact P-value.
  • Drag P-slider: the scale of P-Slider is logarithmic, -log10(P-value).

You may experience a minor slowdown when visualizing TFBS for long sequences (100bp+).

Sequence Tabs & Motif Controls

If there are no parsing errors, one Sequence Tab is created for each sequence. Sequences with markup and highlighting of the motif entry appear in the Output Window, as in the demo above.

You can always change selected sequence colour using Colour Picker. Click on the box with the selected color in the upper left corner of Motif Control to open Colour Picker. It's recommended to use the default colour palette in the last two rows to improve your visual experience. Newly added motifs use the default color palette.

Each Sequence Tab contains additional controls.

  • Close Sequence Tab: clickcloseto hide the sequence data from the Output Window and Entry Table.
  • Copy sequence: click content_copy to copy sequence with markup and highlighting of motif entry. It remains when pasted into popular text editors such as Pages, Word, and LibreOffice.
  • Lock scroll: click lock_open to lock to the horizontal scroll at the current position. Click it again to unlock it.

Horizontal scroll is only available in the Multiple Mode. Therefore, there is no button lock_open in the Single Mode. Read more about View Modes below.

Entry Highlighting

Entry highlighting is a special feature that allows you to better navigate the output and significantly improve visual experience. It visualizes the relationship between data from Motif Controls, the Output Window, and the Entry Table. Hover over the following elements to trigger Entry Highlighting.

  • Motif Control: highlight entries of this motif both in the Output Window and Entry Table.
  • Output Window: hover over a coloured entry in the sequence to highlight the corresponding rows in the Entry Table and found model entry's Motif Control.
  • Entry Table: hover over a row to highlight this entry in the Output Window and corresponding Motif Control.

View Modes

There are two View Modes, Multiple Mode and Single Mode. Multiple Mode is activated automatically after first data submission. Further submissions activate the last used View Mode.

  • Multiple Mode: shows all the entered sequences without line breaks. Horizontal scrolling is available when there is not enough space.
  • Single Mode: shows only one selected sequence in the entire Output Window with line breaks.

You can switch between these modes using the Mode Switch located above the right corner of the text input field or Output Window. You can also click on Mode Button / in Sequence Tabs.


If Single Mode is activated by Mode Switch, the first sequence is automatically selected to be displayed in Single Mode. If Single Mode is activated by Mode Button in the Sequence Tab, that sequence is selected. You can change your selection by clicking another Sequence Tabs.

Entry Table

Detailed data on detected binding sites are provided in the Entry Table at the bottom of the page. Generating the population of this table is the most time-consuming operation.

Clicksyncto invert logo.

The Entry Table has additional controls. Click to select information to be displayed on the table. Selected columns are coloured with white.

The Entry Table contents can be exported using Export Buttons. Available formats: plain text (copy to clipboard), xlsx , PDF , and TSV .

Entry Table contains only sites from displayed sequences. According to the chosen View Mode, the table will contain different sites.
